jQuery MG Gallery Plugin Example Two
Layout with the thumbnail strip above the current image. This uses exchangeType: 'swap' as the image transition type, and is set to a quicker transition speed of .5 seconds. The thumbnail opacity is set to 25%.
  • Click on the main image to load the next image.

  • Any HTML or text can be used in the thumbnailMeta div. If no description is provided, this thumbnailMeta div is hidden.
